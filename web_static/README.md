# 0x01. AirBnB clone - Web static

## Description :house:

Frontend design of each element of the AirBnB_clone.<br>
Steps:
- Create simple HTML static pages
- Style guide
- Fake contents
- No Javascript
- No data loaded from anything
During this project, I learn how to manipulate HTML and CSS languages. HTML is the structure of your page, it should be the first thing to write. CSS is the styling of your page, the design. Indeed, without any structure, you canâ€™t apply any design.
